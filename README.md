# 🏛️ Law Firm Management & Client Portal

Modern hukuk büroları için tasarlanmış, yapay zeka destekli içerik yönetim sistemi ve müvekkil etkileşim platformu. 

### 🚀 Teknolojiler
- **Frontend:** React.js, Vite, TypeScript
- **Styling:** Tailwind CSS, Framer Motion (Premium UI/UX)
- **Backend (BaaS):** Cloud Firestore, Firebase Auth
- **Media:** Cloudinary API (Automated Image Optimization)
- **Icons:** Lucide React

### ✨ Temel Özellikler
- **Dinamik Hesaplama Motoru:** İşçilik alacağı ve harç hesaplamaları için admin panelinden yönetilebilir formül sistemi.
- **AI Chatbot:** Büro lokasyonu ve çalışma alanları hakkında bilgi veren, gerektiğinde avukata yönlendiren asistan.
- **Sosyal Medya Entegrasyonu:** YouTube, Instagram ve LinkedIn içeriklerinin tek bir panelden yönetimi.
- **Sessiz Lüks (Quiet Luxury) Tasarım:** Hukuk dünyasının prestijine uygun Dark/Gold tema mimarisi.
